# root-ui

Root UI built with TypeScript, React and TailwindCSS.

> Each directories will be separated to each packages.

## Prerequisites

- nvm
- pnpm

## Packages

- @root-sphere/root-ui
- @root-sphere/root-ui-tailwind

```bash
brew install nvm # If you don't have nvm
nvm use
npm install -g pnpm
```

## How to use

### Install

```bash
pnpm install
```

## How to run

### Production

```bash
pnpm run build
```

### Dev (with Storybook)

```bash
pnpm run storybook
```

### Build (with Storybook)

```bash
pnpm run build:storybook
```

### Test (with Vitest)

```bash
pnpm run test
```

#### Lint

```bash
pnpm run lint
pnpm run lint:fix
```

#### Prettier

```bash
pnpm run prettier
```
